# Empty Group — Blender Addon

選択したオブジェクト群の**バウンディングボックス中心**にエンプティを自動生成し、ペアレント化でグループ管理する Blender アドオンです。

![Blender](https://img.shields.io/badge/Blender-3.0%2B-orange?logo=blender&logoColor=white)
![Version](https://img.shields.io/badge/version-4.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

---

## 機能

- 選択オブジェクトの**メッシュ形状**からバウンディングボックス中心を計算してエンプティを配置
- グループ化と同時にすべての選択オブジェクトをエンプティの子にペアレント化
- グループ化完了後、**スケールモーダルと同じ操作感**でエンプティサイズを即調整
- **`Ctrl+Shift+G` のコンテキスト分岐**：エンプティ選択時はオプションポップアップを表示
- **階層 / 依存関係 UI**：親・子・孫を最大 3 階層でツリー表示、追加・削除・グループ間移動をその場で編集
- 階層選択（エンプティ含む全選択 / 子のみ選択）
- 名前変更・グループ解除・サイドバーパネルによる統合 UI

---

## インストール

1. [Releases](../../releases) から最新の `empty_group.py` をダウンロード
2. Blender を起動し **編集 → プリファレンス → アドオン → インストール…** を開く
3. `empty_group.py` を選択して「アドオンをインストール」をクリック
4. 一覧に **"Empty Group"** が表示されたらチェックを入れて有効化

> **動作確認済み：** Blender 3.0 以降

## 使い方

### グループ化　`Ctrl + Shift + G`

> エンプティを選択中に押すと → **オプションポップアップ**が開きます（後述）

1. オブジェクトモードでグループ化したいメッシュを複数選択
2. `Ctrl + Shift + G` を押す
3. ダイアログでグループ名・エンプティの種類・表示サイズ・ペアレント設定を確認して **OK**
4. OK 後、自動的に**サイズ変更モード**へ移行

**エンプティの配置位置について：**  
選択オブジェクト全体のバウンディングボックス中心に配置されます。オブジェクト原点ではなく実際のメッシュ形状から計算するため、原点がずれているオブジェクトでも正確な中心に置かれます。

**ダイアログの設定項目：**

| 項目 | 説明 |
|---|---|
| グループ名 | エンプティの名前。同名が存在する場合は `.001` を自動付与 |
| エンプティの種類 | Plain Axes / Arrows / Circle / Cube など 6 種類 |
| 表示サイズ | エンプティの表示サイズ（サイドバーのデフォルト設定を引き継ぎ） |
| ペアレント化 | 選択オブジェクトをエンプティの子にする |
| トランスフォームを維持 | ペアレント後もオブジェクトの位置・回転・スケールを保持する |

---

### サイズ変更モード　`Ctrl + Shift + E`

グループ化直後に自動で移行するほか、エンプティを選択した状態で `Ctrl + Shift + E` を押しても入れます。

| 操作 | 内容 |
|---|---|
| マウスを右へ移動 | サイズを大きくする |
| マウスを左へ移動 | サイズを小さくする |
| 数値キー（`0`〜`9`、`.`） | サイズを数値で直接入力 |
| `Backspace` | 入力を 1 文字削除（全消去でマウス操作に戻る） |
| `Enter` または左クリック | 確定 |
| `Esc` または右クリック | キャンセル（変更前のサイズに戻す） |

操作中はヘッダーバーに現在のサイズと操作ガイドが表示されます。

---

### オプションポップアップ　`Ctrl + Shift + G`（エンプティ選択時）

エンプティを選択した状態で `Ctrl + Shift + G` を押すとポップアップが開きます。

**表示設定**：形状・サイズの変更、インタラクティブサイズ変更ボタン

**階層 / 依存関係**：

```
↑ 親オブジェクト:  ParentObj  [選択]

子オブジェクト（3個）                [選択中を追加 +]
 • MeshA       [選択] [→移動] [✕]
 • MeshB       [選択] [→移動] [✕]
 └─ ChildEmpty [選択] [✕]
      └─ GrandchildMesh  [選択] [→移動] [✕]

[エンプティ含む全選択]  [子のみ選択]
```

| ボタン | 内容 |
|---|---|
| **選択** | そのオブジェクトを選択・アクティブにする |
| **→移動** | 別エンプティへ移動（移動先をダイアログで選択） |
| **✕** | グループから外す（位置は維持） |
| **選択中を追加** | 現在選択中のオブジェクトをこのグループに追加 |

---

### 階層選択

| オペレータ | 内容 |
|---|---|
| エンプティ含む全選択 | アクティブエンプティ＋全子孫を選択 |
| 子のみ選択 | 子孫オブジェクトのみ選択（エンプティは含まない） |

---

### 名前変更　`Ctrl + Shift + R`

エンプティを選択した状態で押すと名前変更ダイアログが開きます。

---

### グループ解除　`Ctrl + Shift + U`

エンプティを選択した状態で押すと、子オブジェクトをすべて独立させます（ワールド位置は維持）。エンプティ自体を削除するかどうかも選択できます。

## ショートカット一覧

| キー | 機能 | 使用条件 |
|---|---|---|
| `Ctrl + Shift + G` | グループ化 | メッシュを 1 つ以上選択中 |
| `Ctrl + Shift + G` | オプションポップアップ | **エンプティを選択中** |
| `Ctrl + Shift + E` | サイズ変更モード | エンプティを選択中 |
| `Ctrl + Shift + R` | 名前変更 | エンプティを選択中 |
| `Ctrl + Shift + U` | グループ解除 | 子を持つエンプティを選択中 |

---

## サイドバーパネル

`N` キー → **"Empty Group"** タブ

| セクション | 内容 |
|---|---|
| デフォルト設定 | グループ化時のデフォルトサイズ・種類をシーンごとに保存 |
| グループ化ボタン | 選択オブジェクト数を表示しながらワンクリックでグループ化 |
| 選択エンプティの操作 | 表示サイズ・種類をスライダーで直接変更、各操作ボタンも表示 |

---

## 注意事項

- オブジェクトモードでのみ動作します
- 選択オブジェクトにメッシュ・カーブ・テキスト・アーマチュアなどが含まれていない場合はグループ化できません
- ペアレント化には Blender 組み込みの `parent_set` を使用しています。既存のペアレント関係は上書きされます
